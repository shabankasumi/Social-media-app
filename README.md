# 📱 Social Media App

A simple full-stack social media application where users can create posts and interact with content.

## 🚀 Tech Stack

* React.js
* Node.js / Express
* MongoDB

## ✨ Features

* User authentication
* Two-Factor Authentication (2FA) with email (Nodemailer)
* Create and like posts
* Basic user interaction system
* REST API integration

## ⚙️ Run

```bash
cd backend && npm install && npm start
cd frontend && npm install && npm start
```
